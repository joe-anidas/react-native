DESIGN AN ANDROID APPLICATION USING CORDOVA FOR A USER LOGIN SCREEN WITH USERNAME, PASSWORD, RESET BUTTON AND A SUBMIT BUTTON. ALSO, INCLUDE HEADER IMAGE AND A LABEL. USE LAYOUT MANAGERS.

AIM:
	





PROCEDURE:

Install Cordova globally:

npm install -g cordova

Create a new Cordova project:

cordova create myApp

Navigate to the project directory:

cd myApp

Add Android platform:

cordova platform add android

Build the Cordova project:

cordova build android

Edit index.html, styles.css, and index.js according to your requirements: Create the login UI, define layout with HTML and CSS, and add functionality using JavaScript.

Run the app on an emulator or device:

cordova run android

//app.js

document.addEventListener('deviceready', function() {
    const usernameField = document.getElementById('username');
    const passwordField = document.getElementById('password');
    const resetButton = document.getElementById('reset');
    const submitButton = document.getElementById('submit');

    resetButton.addEventListener('click', function() {
      usernameField.value = '';
      passwordField.value = '';
    });
 
    submitButton.addEventListener('click', function() {
      const username = usernameField.value;
      const password = passwordField.value;
     
      if (username && password) {
        alert('Login successful!');
      } else {
        alert('Please enter both username and password.');
      }
    });
  });
