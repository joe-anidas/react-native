DESIGN AND DEVELOP AN ANDROID APPLICATION USING APACHE CORDOVA TO FIND AND DISPLAY THE CURRENT LOCATION OF THE USER.


AIM:
	To design and develop an android application using Apache Cordova to find and display the current location of the user. 
PROCEDURE:
Install Apache Cordova globally using npm.
>npm install -g cordova
Create a new Cordova project for the location app.
>cordova create LocationApp com.example.locationapp LocationApp
Change directory to the newly created project folder.
>cd LocationApp
Add Android platform to your Cordova project.
>cordova platform add android
Install the geolocation plugin to access the device's location.
>cordova plugin add cordova-plugin-geolocation
Build the project for Android platform.
>cordova build android
Modify index.html, index.css, and index.js for UI and location functionality.
Run the app on an Android emulator or device.
>cordova run android




CODE:

//index.html
<!DOCTYPE html>
<html>
<head>
    <title>Location App</title>
    <link rel="stylesheet" type="text/css" href="css/index.css">
    <script type="text/javascript" src="cordova.js"></script>
    <script type="text/javascript" src="js/index.js"></script>
</head>
<body>
    <div class="container">
        <h2 class="heading">Location App</h2>
        <button class="button" onclick="getLocation()">Get Location</button>
        <div id="location" class="location-info"></div>
    </div>
</body>
</html>

//index.js
function getLocation() {
    navigator.geolocation.getCurrentPosition(onSuccess, onError);}

function onSuccess(position) {
    var latitude = position.coords.latitude;
    var longitude = position.coords.longitude;
    document.getElementById('location').innerHTML = 'Latitude: ' + latitude + '<br>Longitude: ' + longitude;}

function onError(error) {
    alert('Error occurred. Code: ' + error.code + '\nMessage: ' + error.message);}
